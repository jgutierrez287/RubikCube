# RubikCube
In processing sketchbook I created a 3D model of a 3x3x3 rubik cube. The cube sides can be rotated using the keyboard.

Controls:
U-Up face
D-Down face
R-Right face
L-Left face
B-Back face
F-Front face

u-Up prime face
d-Down prime face
r-Right prime face
l-Left prime face
b-Back prime face
f-Front prime face
